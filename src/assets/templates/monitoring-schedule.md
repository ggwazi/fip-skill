## Daily Home Monitoring Checklist

**Record daily:**
- [ ] Temperature (normal: 38.0-39.2°C)
- [ ] Appetite (poor / fair / good / excellent)
- [ ] Activity level (lethargic / quiet / normal / playful)
- [ ] Injection site condition
- [ ] Any concerning signs

## Signs Requiring Immediate Veterinary Attention

- [ ] Persistent fever (>39.5°C)
- [ ] Complete loss of appetite (>24 hours)
- [ ] Difficulty breathing
- [ ] Neurological changes (seizures, disorientation, paralysis)
- [ ] Sudden vision changes
- [ ] Severe injection site reaction (large swelling, open wound, extreme pain)
- [ ] Collapse or extreme lethargy
